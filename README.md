# create-drupal-camp-schedule

Interactive CLI to scaffold a conference schedule app from the [drupal-camp-schedule](https://github.com/bison-digital/drupal-camp-schedule) template.

## Usage

```sh
npm create drupal-camp-schedule
```

Or with npx:

```sh
npx create-drupal-camp-schedule
```

The CLI will prompt you for:

- Project directory
- Conference name and short name
- Conference days (dates and labels)
- Colour theme (Ocean, Sunset, Forest, Slate)
- Footer credit text and URL

It then clones the template, applies your configuration, and optionally installs dependencies.
